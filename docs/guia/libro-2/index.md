# ⚙️ Libro II · Automatización básica

¡Enhorabuena!

Si has llegado hasta aquí significa que ya has superado la fase de supervivencia. Tienes una base funcional, herramientas suficientes y conoces los conceptos básicos del modpack.

Ahora comienza el verdadero corazón de **SteamPunk**.

En este libro aprenderás a utilizar **Create** para construir tus primeras automatizaciones, conectar máquinas entre sí y transformar una simple base en una pequeña fábrica capaz de producir recursos de forma automática.

No te preocupes si nunca has utilizado este mod antes. Avanzaremos paso a paso, explicando los componentes más importantes y enseñándote cómo pensar a la hora de diseñar una fábrica eficiente.

# 🎯 Objetivos del Libro II

Cuando completes este libro serás capaz de:

- ✅ Generar y transportar energía mecánica.
- ✅ Construir líneas de producción sencillas.
- ✅ Automatizar recursos básicos.
- ✅ Organizar correctamente una fábrica.
- ✅ Comprender el funcionamiento general de Create.
- ✅ Preparar tu base para proyectos mucho más ambiciosos.

---

# 💡 Consejo del Codex

No tengas prisa.

La mayoría de jugadores intentan construir enormes fábricas demasiado pronto.

Es mucho mejor aprender cómo funciona cada automatización y ampliar la base poco a poco.

Una buena planificación hará que el resto del modpack resulte mucho más sencillo.
