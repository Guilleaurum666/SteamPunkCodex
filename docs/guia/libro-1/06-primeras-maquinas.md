# ⚙️ Primeras máquinas

Hasta ahora todo lo has hecho manualmente.

A partir de este momento empieza la verdadera experiencia de SteamPunk: **la automatización**.

No hace falta construir una megafábrica desde el principio. Empieza con pequeñas máquinas que te ahorren trabajo.

---

# ⚙️ Tu primer objetivo: Create

El mod principal del servidor es **Create**.

Con él podrás construir:

- ⚙️ Molinos.
- 🔄 Cintas transportadoras.
- 🏭 Máquinas automáticas.
- 🚂 Trenes.
- 📦 Sistemas de transporte de objetos.
- 🌾 Granjas automáticas.

Todo funciona mediante energía mecánica.

---

# 🌬️ Genera energía

Al principio la forma más sencilla es utilizar un **Water Wheel**.

Ventajas:

- 💧 Barato.
- 🔧 Fácil de fabricar.
- ♻️ Funciona continuamente.
- ⚙️ Suficiente para las primeras máquinas.

Más adelante podrás utilizar sistemas mucho más potentes.

---

# ⚙️ Transmite el movimiento

La energía mecánica se mueve mediante:

- 🔩 Shafts.
- ⚙️ Cogwheels.
- 🔀 Gearboxes.
- 📐 Chain Drives.

No tengas miedo de experimentar.

Mover la potencia correctamente forma parte del aprendizaje de Create.

---

# 📦 Automatiza tareas pequeñas

No intentes automatizar todo de golpe.

Empieza por cosas sencillas como:

- 🌾 Recolectar cultivos.
- 🪨 Triturar piedra.
- 🌾 Procesar materiales.
- 📦 Transportar objetos.
- 🪵 Automatizar madera.

Cada pequeña automatización te ahorrará muchísimo tiempo.

---

# 🏭 Construye con espacio

Un error muy común es hacer una fábrica demasiado pequeña.

Deja espacio para:

- Nuevas máquinas.
- Almacenes.
- Cintas.
- Trenes.
- Decoración.

En SteamPunk las fábricas crecen constantemente.

---

# 💡 Consejo del Codex

No copies fábricas gigantes de Internet.

Entender cómo funciona cada mecanismo es mucho más útil que construir algo enorme sin saber por qué funciona.

Empieza pequeño.

Aprende.

Y mejora poco a poco.

---

# ✅ Antes de continuar...

Procura tener:

- ✅ Tu primer sistema de energía.
- ✅ Algunos ejes y engranajes.
- ✅ Una máquina sencilla funcionando.
- ✅ Espacio reservado para ampliar la fábrica.
