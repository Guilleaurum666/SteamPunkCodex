# 🏡 Primer refugio

Ya tienes recursos, comida y un lugar donde establecerte.

Ahora toca construir una base que te permita crecer durante todo el Libro I sin tener que mudarte a las pocas horas.

---

# 📍 Elige una buena ubicación

Antes de colocar el primer bloque, piensa en el futuro.

Una buena base debería tener cerca:

- 🌊 Agua.
- 🌲 Bosques.
- ⛰️ Cuevas.
- 🌾 Espacio para cultivos.
- 🚂 Sitio para una estación de tren.
- 🏭 Mucho terreno libre para ampliar la fábrica.

No hace falta construir una mansión.

Lo importante es tener espacio.

---

# 🏠 No construyas una casa enorme

Uno de los errores más comunes es dedicar horas a decorar la casa inicial.

En SteamPunk tu base crecerá constantemente.

Empieza con algo sencillo y funcional.

Más adelante podrás construir una auténtica ciudad industrial.

---

# 📦 Organiza el almacenamiento

Aunque todavía tengas pocos recursos, merece la pena mantener cierto orden.

Puedes separar los cofres por categorías:

- 🪵 Madera.
- 🪨 Piedra.
- ⛏️ Minerales.
- 🌾 Comida.
- ⚙️ Componentes de Create.
- 📦 Objetos variados.

Cuanto antes empieces a organizarte, más fácil será ampliar la base.

---

# 🔥 Automatiza lo básico

Antes de avanzar intenta disponer de:

- 🔥 Uno o varios hornos.
- 💧 Un cubo de agua.
- 🌾 Una pequeña granja.
- 🐄 Algunos animales cerca.
- 🛏️ Una cama.

Estos recursos harán mucho más cómoda la progresión.

---

# 🚂 Piensa en la futura estación

Aunque todavía no construyas trenes, deja un espacio libre para ellos.

Más adelante agradecerás no tener que derribar media base para colocar las vías.

Una zona amplia y recta suele funcionar muy bien.

---

# 💡 Consejo del Codex

Las mejores bases no son las más bonitas.

Son las que permiten ampliar la fábrica sin tener que reconstruirlo todo cada pocas horas.

Piensa siempre en el espacio que necesitarás dentro de unos capítulos.

---

# ✅ Antes de continuar...

Procura tener:

- ✅ Una base donde pasar la noche.
- ✅ Cofres organizados.
- ✅ Hornos funcionando.
- ✅ Una pequeña granja.
- ✅ Espacio reservado para futuras máquinas.
