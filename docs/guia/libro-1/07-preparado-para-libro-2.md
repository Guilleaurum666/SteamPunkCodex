# 🚀 Preparado para el Libro II

¡Enhorabuena!

Has completado los primeros pasos de tu aventura en SteamPunk.

Ahora ya no eres un superviviente... estás a punto de convertirte en un verdadero ingeniero.

---

# 📚 Lo que has aprendido

Durante este primer libro has conseguido:

- 🌍 Elegir un buen lugar para vivir.
- 🌳 Reunir los recursos básicos.
- 🍖 Conseguir una fuente estable de comida.
- 🏡 Construir tu primera base.
- ⛏️ Fabricar las herramientas necesarias.
- ⚙️ Dar los primeros pasos con Create.

Ya tienes una base sólida sobre la que construir todo lo demás.

---

# 🎯 Comprueba tu progreso

Antes de continuar sería recomendable tener:

- ✅ Una base organizada.
- ✅ Espacio libre para ampliar la fábrica.
- ✅ Varias pilas de hierro.
- ✅ Comida suficiente.
- ✅ Un sistema básico de energía mecánica.
- ✅ Las primeras máquinas de Create funcionando.
- ✅ Materiales almacenados y ordenados.

No es obligatorio, pero hará que el siguiente libro sea mucho más cómodo.

---

# 🏭 A partir de ahora...

En el Libro II empezará la verdadera automatización.

Aprenderás a:

- ⚙️ Procesar materiales automáticamente.
- 🏭 Crear líneas de producción.
- 📦 Transportar objetos entre máquinas.
- 🔄 Mejorar el rendimiento de tu fábrica.
- 🚂 Preparar el camino hacia la red ferroviaria.

Cada capítulo añadirá nuevas piezas a una fábrica mucho más grande.

---

# 💡 Consejo del Codex

No intentes hacerlo todo perfecto.

Las fábricas evolucionan constantemente.

Es completamente normal desmontar máquinas, reorganizar edificios o ampliar la base conforme aprendes nuevas técnicas.

La mejor fábrica no es la más bonita.

Es la que sigue creciendo.

---

Ya conoces los fundamentos del servidor y estás preparado para empezar a automatizar de verdad.

El siguiente paso será convertir una pequeña base en una auténtica fábrica industrial.
