# 🏠 Bienvenido a SteamPunk Codex

¡Bienvenido a **SteamPunk Codex**!

Esta guía está pensada para ayudarte a disfrutar del modpack **SteamPunk [LPS] v19**, tanto si es tu primera partida como si ya tienes experiencia con Minecraft modificado.

Antes de comenzar la guía, dedica unos minutos a preparar correctamente el juego. Una buena configuración evitará la mayoría de problemas de rendimiento.

---

# 💻 RAM recomendada

SteamPunk es un modpack grande y necesita bastante memoria para funcionar correctamente.

Recomendamos asignar aproximadamente el **75% de la memoria RAM disponible**, dejando siempre varios gigabytes libres para Windows y el resto de programas.

| RAM del PC | RAM para Minecraft |
|------------|-------------------:|
| **8 GB** | **6 - 6,5 GB** |
| **16 GB** | **11,5 - 12 GB** |
| **32 GB** | **29 - 30 GB** |

> ⚠️ No asignes toda la memoria del ordenador a Minecraft. El sistema operativo también necesita RAM para funcionar correctamente.

---

# ☕ Utiliza Java de 64 bits

SteamPunk necesita una versión de **Java de 64 bits**.

Si utilizas un launcher moderno normalmente se instalará automáticamente.

Si experimentas problemas al iniciar el juego, lo primero que debes comprobar es que estás utilizando la versión correcta de Java.

---

# 🎮 Rendimiento

Algunos consejos para mejorar los FPS:

- 🖥️ Juega en pantalla completa.
- 🎮 Cierra programas que no estés utilizando.
- 🌿 Reduce la distancia de renderizado si tienes pocos FPS.
- 💡 Evita utilizar shaders muy pesados durante las primeras horas.
- ⚙️ Mantén actualizados los drivers de tu tarjeta gráfica.

SteamPunk ya incluye varios mods de optimización, por lo que normalmente no será necesario instalar ninguno adicional.

---

# 🌐 Conexión a Internet

Aunque SteamPunk puede jugarse con una conexión modesta, es recomendable utilizar una conexión estable.

Si puedes elegir entre Wi-Fi y cable Ethernet, el cable siempre ofrecerá una experiencia más estable.

# 💡 Un último consejo

SteamPunk no está pensado para completarse en pocos días.

Disfruta del proceso, construye con calma, experimenta con las máquinas y colabora con otros jugadores.

Gran parte de la diversión está en ver cómo tu pequeña base acaba convirtiéndose en una enorme fábrica conectada mediante trenes.
