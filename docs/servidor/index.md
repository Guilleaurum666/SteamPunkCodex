# 🏰 Información del servidor

Bienvenido al servidor de **SteamPunk [LPS]**.

Antes de comenzar tu aventura, merece la pena dedicar unos minutos a conocer cómo está organizado el mundo y cuál es la filosofía del servidor. Esto te evitará muchos errores durante las primeras horas.

Aquí encontrarás toda la información relacionada con el funcionamiento del servidor, las recomendaciones para construir tu base y la forma en la que convivimos entre jugadores.

---

## 🚂 ¿Qué hace diferente a este servidor?

SteamPunk está pensado para jugar a largo plazo.

El objetivo no es correr hasta el final del modpack, sino construir una fábrica cada vez más grande, conectar bases mediante trenes y colaborar con el resto de jugadores.

La exploración, la automatización y la creación de infraestructuras forman parte de la experiencia.
